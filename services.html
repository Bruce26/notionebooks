Fixed Layout

Fixed Layout is the term used for a special type of eBook that preserves page design, allowing us to create an eBook that maintains the same layout as your print book. Instead of the Fluid Layout used for most eBooks, Fixed Layout allows us to "fix" images and text to exact spots on each page. This prevents images and text from shifting around and lets us create beautiful two-page spreads.

What kind of books should use Fixed Layout?

Illustrated children's books, cook books, coffee table books, graphic novels, technical manuals and any book that relies on a heavily-designed, static presentation, full-page images, or text that overlaps with images.

Where can I sell my Fixed Layout eBook?

Currently, Apple’s iBookstore and Amazon (for the Kindle Fire exclusively) are the only places where authors and independent publishers can sell Fixed Layout eBooks.
